Quick Setup FAQ

How do I install and use the correct configuration profile?

Download the correct profile for your device + aircraft + simulator, then load it in your configuration tool (commonly MobiFlight). Always connect the device before launching the tool.

Where are the official installation guides?

Use the official ProDeskSim installation guides here: Installation Guides.

MSFS users: do I need the WASM module?

Many setups require the MSFS WASM module for proper simulator connectivity. If your guide mentions it, install it from within your configuration tool and then re-launch the sim.

How do I leave a review?

Use the button in the reviews section to write a store review. Reviews help other simmers and also help us improve documentation and profiles.
